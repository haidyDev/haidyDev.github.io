# SQL Orders Analysis

SQL-ajattelun demo, joka näyttää relaatioiden hyödyntämistä kolmessa vaiheessa:

1. `JOIN` asiakkaat ja tilaukset.
2. Laske asiakkaiden kokonaisostot `GROUP BY` -kyselyllä.
3. Nosta esiin eniten liikevaihtoa tuottavat tuotteet.

Demo on täysin staattinen ja käyttää mock-tulostaulukoita havainnollistamiseen.
